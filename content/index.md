# 🚀 ProjectZ

Sistema de documentação com suporte a **markdown**, **slides** com reveal.js e **code trees**.

## Funcionalidades

- 📝 Páginas escritas em Markdown
- 📂 Menu lateral gerado automaticamente pela estrutura de pastas
- 🎥 Slides com reveal.js
- 🌳 Visualizador de estrutura de código estilo VS Code
- 📊 Formulários integrados com Google Sheets

## Começando

Navegue pelo menu lateral para explorar as seções.
